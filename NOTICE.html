Demo Maker HOW TO

Base software requirements

Vagrant and Virtualbox programs are required to install demos on your computer. To install these programs you should follow boxes installation guide.

Demo Maker usage

To install the demo maker, use the install(.bat) script in this folder. It only has to be run once.

Once you have completed installation, you can use four scripts:

start(.bat)
Should be used to start the box. (it will create the box if the box doesn't already exist)
stop(.bat)
Stops the box and saves its state.
destroy(.bat)
Destroys the box.
reload(.bat)
Reload the data in the box
update(.bat)
Recreates and updates the box
boost
On Linux and Mac only, allow to customize box memory and CPU, php memory and ElasticSearch memory

To run these scripts on MacOs, you will have to right click them and choose Open in the menu.

Products can be imported inside the demo by editing the files in the data folder. If you have been modifying files in the data folder and you want the modification to appear in the PIM you should use reload(.bat)

Once your demo has been started, it will be accessible at the following URLs :

  • Akeneo PIM: http://localhost:8072

The following users will be available:
